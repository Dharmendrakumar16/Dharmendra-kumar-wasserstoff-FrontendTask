## Steps to Run/Build the Project

```bash
git clone https://github.com/Dharmendrakumar16/Dharmendra-kumar-wasserstoff-FrontendTask.git
cd wasserstoff-FrontendTask
- To run the project
npm install
npm run dev

npm run build
```
